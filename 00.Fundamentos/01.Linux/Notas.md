# Linux Introduccion

## Conocimiento previo/Entendimiento previo
    Antes de empesar a estudiar creia que linux era un sistema operativo como windows pero gratuito y que cualquiera podia modificarlo

## Qué es
    Es un nucleo o mas conocido como Kernel, creado por Linus Torvald el kernel de linux es un sistema que conecta el hardware con el software

## Qué problema resuelve
    Anterior a Linux existia Unix, como sistema pero este fue comprado y privatizado evitandoa si su uso por usuarios comunes, ahi entra linux, que es un kernel hecho a partir de Minix, que es una version de aprendisaje de Unix

## Conceptos clave
- Kernel
- Distro/Distribucines

    Un sistema operativo como tal.
