Esta carpeta busca englobar los fundamentos de los temas bases para DevOps y Data.

Los documentos cuyo nombre es Notas.md poseen la siguiente estructura base sujeto a posibles modificaciones:

    # Tema

    ##Conocimiento previo/Entendimiento previo

    ## Qué es
    Explicación con palabras propias.

    ## Qué problema resuelve
    Por qué existe esto.

    ## Conceptos clave
    - Concepto 1
    - Concepto 2

    ## Qué NO es
    Errores comunes.

    ## Dudas / cosas para revisar
       

Esta disposicion comple el proposito de facilitar la lectura del material para su estudio.
El directorio de fundamentos va a contener varios conceptos primarios
Cada archivo Notas.md no busca ser documentación exhaustiva ni académica, sino un registro personal de comprensión progresiva del tema, escrito con palabras propias y ejemplos mentales.
Estas notas pueden cambiar con el tiempo a medida que avanze con el estudio y entendimiento de cada tema

Ademas de archivos Notas.md pueden haber otros archivos .md cuya disposicion puede ser diferente a las notas
A medida que se vayan agregando estos archivos su presentacion y proposito seran detallados en este documento
